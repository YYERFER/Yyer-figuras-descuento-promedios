Calcular el Área de un Cuadrado

El cuadrado ◻ es una figura geométrica caracterizada por ser un tipo de paralelogramo con cuatro lados de igual longitud y paralelos entre sí. Para calcular el área de un Cuadrado, tenemos la siguiente formula:

A= b x h

donde A es el área, 'b' es la base y 'h' la altura

"b=h"

Calcular el Área de un Triangulo Rectángulo

El triángulo rectángulo ◺ es aquel que tiene un ángulo interior que es recto, es decir, mide 90º. Para calcular el área de un Triangulo Rectángulo se tiene la siguiente formula:
